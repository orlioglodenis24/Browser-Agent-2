# Browser Agent 2

## Описание
Мультиагентная система для автоматизации браузера с поддержкой AI и компьютерного зрения.

## Быстрый старт: переменные окружения
Для работы некоторых функций требуется файл `.env` с вашим API-ключом DeepSeek.

1. Получите ключ на https://platform.deepseek.com/ (или у вашего администратора).
2. Создайте файл `.env` в корне проекта и добавьте строку:
  ```env
  DEEPSEEK_API_KEY=ваш_ключ_сюда
  ```
3. Без этого ключа часть AI-функций работать не будет.

## Требования
- Python 3.8–3.11 (рекомендуется 3.11)
- pip
- macOS/Linux/Windows

## Установка
1. Клонируйте репозиторий или скачайте проект.
2. Перейдите в папку проекта:
   ```sh
   cd browser-agent2
   ```
3. Установите зависимости:
   ```sh
   pip install -r requirements-free.txt
   ```

## Запуск
- **Полная версия:**
  ```sh
  python3 main.py
  ```
- **Упрощённая версия:**
  ```sh
  python3 main_simple.py
  ```

## Использование
1. При запуске введите задачу для агента (например: "Найди рецепт борща").
2. Следуйте инструкциям в терминале.

## Примечания
- Для работы некоторых функций требуется установленный [Ollama](https://ollama.com/) (локальные AI-модели).
- Скриншоты и результаты сохраняются в папке проекта.
- Для Playwright может потребоваться установка браузеров:
  ```sh
  playwright install
  ```
