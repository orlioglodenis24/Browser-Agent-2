# Browser Agent 2

## Описание
Мультиагентная система для автоматизации браузера с поддержкой AI и компьютерного зрения.

## Требования
- Python 3.8–3.11 (рекомендуется 3.11)
- pip
- macOS/Linux/Windows

## Установка
1. Клонируйте репозиторий или скачайте проект.
2. Перейдите в папку проекта:
   ```sh
   cd browser-agent2
   ```
3. Установите зависимости:
   ```sh
   pip install -r requirements-free.txt
   ```

## Запуск
- **Полная версия:**
  ```sh
  python3 main.py
  ```
- **Упрощённая версия:**
  ```sh
  python3 main_simple.py
  ```

## Использование
1. При запуске введите задачу для агента (например: "Найди рецепт борща").
2. Следуйте инструкциям в терминале.

## Примечания
- Для работы некоторых функций требуется установленный [Ollama](https://ollama.com/) (локальные AI-модели).
- Скриншоты и результаты сохраняются в папке проекта.
- Для Playwright может потребоваться установка браузеров:
  ```sh
  playwright install
  ```
